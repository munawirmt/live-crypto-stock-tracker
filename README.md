# Advanced AI-Enabled Live Crypto & Stock Portfolio Tracker

An automated quantitative financial tracking system that fetches real-time cryptocurrency and stock prices, performs automated trend analysis, and dispatches instant email alerts when custom profit targets or stop-loss thresholds are breached.

## 🚀 Key Features
- **Real-Time Data Integration:** Zero-cost live market data fetching via `yfinance` API.
- **AI Trend Analysis & Signals:** 7-day statistical trend computation giving `STRONG BUY`, `HOLD`, or `WATCH CLOSELY` signals.
- **Multi-Asset Dashboard:** Tracks global crypto assets (USD) and Indian Equities (NSE/BSE) simultaneously with automated currency conversions.
- **Serverless Automation:** Scheduled deployment via GitHub Actions (Cron CI/CD Workflow) running 24/7 with zero infrastructure cost.
- **Secure Credentials:** Enterprise-grade security utilizing GitHub Repository Secrets to protect sensitive API and SMTP configurations.

## 🛠️ Tech Stack
- **Language:** Python 3.10
- **Libraries:** `yfinance`, `requests`, `smtplib`, `time`
- **Automation:** GitHub Actions (CI/CD)
